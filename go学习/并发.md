# goroutine
## 什么是goroutine？
**Goroutine** 是 Go 语言中实现并发编程的核心机制，它是一种轻量级的线程，由 Go 运行时（runtime）调度和管理。
## goroutine创建方法
### 1.启动普通函数（已创建的函数）
```go
package main

import (
	"fmt"
	"time"
)

func sayHello() {
	fmt.Println("hello goroutine")
}

func main() {
	go sayHello()

	time.Sleep(time.Second) //加休眠，因为主程序也是goroutine，main结束了程序就直接结束了，休眠一点时间给子goroutine执行
	fmt.Println("main end")
}
```
### 2.启动匿名函数（未创建）
```go
package main

import (
	"fmt"
	"time"
)

func main() {
	go func() {
		fmt.Println("匿名 goroutine 执行")
	}()

	time.Sleep(time.Second)
}
```
### 3. 带参数的goroutine
```go
package main

import (
	"fmt"
	"time"
)

func main() {
	name := "Go"

	go func(s string) {
		fmt.Println("hello", s)
	}(name)//推荐传入外部参数

	time.Sleep(time.Second)
}
```
# waitGroup
## 什么是waitGroup
`sync.WaitGroup` 是 Go 语言标准库中用于**同步**（Synchronization）的最常用工具。它的功能非常纯粹：**等待一组 Goroutine 全部执行完毕。**

你可以把它想象成一个“倒计时计数器”：

1. 你每启动一个任务，就让计数器 **+1**。
    
2. 每个任务完成后，就让计数器 **-1**。
    
3. 主程序（或某个协程）调用“等待”方法，如果计数器不为 0，它就**阻塞**在那里，直到计数器变成 0 为止。
